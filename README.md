# Local Quick Planner

Local Quick Planner is a lightweight Microsoft Plannerâ€“style board that runs entirely in your browser. All data lives in `localStorage` so your tasks stay on your machine.

## Features
- **My Day** view with To Do, In Progress, Done columns
- **My Tasks** Kanban board with Ideas, Backlog, In Progress, Done
- Drag & drop powered by [dnd-kit](https://dndkit.com)
- State management via [Zustand](https://github.com/pmndrs/zustand)
- Export / Import JSON and clear all data

## Tech Stack
- Next.js (App Router) + React + TypeScript
- Tailwind CSS
- Zustand
- dnd-kit

## Quick Start
```bash
npm install
npm run dev
```
Then open http://localhost:3000 in your browser.
